# 本项目使用教程

## 上位机-SecureCRT

请自行百度下载并安装该软件。

## 跑代码

- 把**1-刷外部FLASH程序（烧录emXGUI资源文件）-串口YMODEM**的代码下载到开发板；
- 复位开发板运行代码；
- 按KEY2此时LED2绿灯亮起（开始擦除FLASH）；
- 漫长的等待，等待LED2绿灯灭掉（擦除完成）；
- 按下KEY1,此时等上位机配置好就开始传输文件了；
- 开始配置上位机（跟着图片操作就OK）：
